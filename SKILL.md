# SKILL.md - 竞品价格监控系统

## 触发词
竞品监控、价格监控、竞争分析、电商运营、监控竞品、对手价格

## 能力说明

这是一个**竞品价格监控**的 QClaw Skill，帮助卖家自动追踪竞品价格变化。

### 核心功能
- `dashboard` — 查看所有监控商品的仪表盘（最低价、趋势、提醒）
- `refresh` — 刷新所有商品价格（模拟价格波动）
- `detail <商品>` — 查看单个商品的详细竞品分析
- `alerts` — 检查当前价格提醒
- `report` — 导出分析报告
- `status` — 查看系统状态

### 工作流程
1. 加载已监控的商品列表（支持多个商品）
2. 获取各竞品的最新价格数据
3. 计算价格趋势和变化幅度
4. 生成可视化报告和购买建议
5. 触发阈值提醒

### 数据存储
- 商品信息：`competitor_data.json`
- 历史记录：30天价格快照
- 无需外部数据库

### 注意事项
- 当前版本使用模拟数据演示完整功能
- 真实数据采集依赖浏览器自动化（xbrowser）
- 价格提醒阈值默认5%，可通过配置调整
